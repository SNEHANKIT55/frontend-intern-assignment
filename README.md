A desktop-only, pixel-accurate quiz UI implemented in React + TypeScript + Tailwind CSS based on the provided Figma screens (question flow, progress bar, navigation, and final score).

---

## Tech stack
- React + TypeScript
- Vite
- Tailwind CSS
- (Optional) Framer Motion placeholders for animation
- Deployed on Vercel

---

## Features implemented
- Pixel-accurate desktop UI based on Figma screenshots
- Multi-step quiz flow with:
  - Progress bar with segments
  - Full-width option buttons with hover/selected states
  - Prev / Next navigation and Finish flow
  - Final score screen with large percentage
- Accessibility considerations:
  - Semantic HTML (`main`, `header`, `section`, etc.)
  - Keyboard focus states
  - `aria-pressed` on option buttons and accessible labels
- Build scripts and production configuration for Vercel
